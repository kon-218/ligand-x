# Ligand-X React Frontend

Modern React/Next.js frontend for the Molecular Structure Processing Application.

## Features

- **Viewer-Centric Design**: Large molecular viewer with collapsible side panel
- **Modern UI**: Built with React, Next.js, TypeScript, and Tailwind CSS
- **Smooth Animations**: Framer Motion for panel transitions
- **State Management**: Zustand for client state, React Query for server state
- **3Dmol.js Integration**: Interactive molecular visualization

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
